# Calculadora
 Calculadora construida em C# com possibilidade de exibir o historico.
 
 <p>faz calculos basicos como:</p>
 <div>- Adição</div>
 <div>- Subtração</div>
 <div>- Multiplicação</div>
 <div>- Divisão</div>
 <div>- Porcentagem</div>
